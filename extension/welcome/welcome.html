Welcome to Bolt Capture

Intercept browser downloads and send them to Bolt.

1

Disable the Save As dialog

For seamless interception, turn off Ask where to save each file in Chrome's download settings. This lets Bolt capture downloads without a dialog popping up first.

2

Configure the extension

Click the Bolt Capture icon in your browser toolbar to set the server URL and auth token. You can copy the token from the Bolt desktop app (Settings → Auth Token → Copy) or find it in ~/.config/bolt/config.json.

You're all set. Close this tab and start downloading.
